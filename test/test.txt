1. cd *название директории*
2. ~ - указывает на домашнюю директорию
   . - обозначает текущую директорию
   .. - обозначает родительскую директорию
3. удаляет файл или папку
4. перемещение между папками
5. копирует файлу или папку
6. Перемещяет(или переименновывает) файлы или директории
7. mv file1.txt(это старое имя) file2.txt(это новое имя)
8. mv file1.txt(имя файла) dir/(путь)
9. рекурсивно выводит дерево каталогов
10. выводит список файлов в обратном порядке
11. утилита для отслеживания и введения истории изменения файлов в проекте
12. создает новый репозиторий git
13. git checkout -b *название новой ветки*
14. git checkout *название существующей ветки*
15. git merge *название ветки*
16. позволяет удалять отдельный файлы или группы файлов
17. позволяет отменить некоторые существующие коммиты и записать некоторые новые коммиты
